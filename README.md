## About The Project

Welcome to your day, reimagined.

Our dynamic daily planner isn't just a place to jot down tasks—it's your personal command center, evolving with you. Whether you're chasing deadlines, carving out quiet moments, or balancing chaos with calm, this planner adapts in real-time to help you own every hour.

Smart scheduling. Seamless priorities. Gentle nudges when you need them—and space when you don’t.

More than planning. It’s daily choreography.

<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jonathan4648/Senior-Project.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Change git remote url to avoid accidental pushes to base project
   ```sh
   git remote set-url origin jonathan4648/Senior-Project
   git remote -v # confirm the changes
   ```
